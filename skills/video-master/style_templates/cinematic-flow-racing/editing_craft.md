# Editing Craft

## 剪辑原则

剪辑要像心理压力曲线，而不是平均分镜表。前半段用断裂和压缩制造不安，临界段用快切堆叠触觉信息，暴风眼段用突然留白制造反差，结尾用更长、更稳定的镜头完成心流释放。

## 可复用技巧

| 技巧 | 用法 | 目的 |
| --- | --- | --- |
| 硬切 | 在机械细节、眼神、仪表、手部动作之间直接切换 | 让观众感到没有缓冲 |
| 冲击切 | 在声音峰值、光闪、动作撞点处切下一镜 | 制造临界压力 |
| 声画错位 | 下一镜的声音提前进入，或上一镜的声音延迟离开 | 让镜头连接更紧 |
| J-cut/L-cut | 口播或 SFX 先于画面进入/滞后退出 | 保持心理连贯 |
| 节奏断崖 | 快切后突然进入长镜头和低声场 | 形成暴风眼 |
| 重复意象回环 | 反复出现眼神、红灯、手、仪表、雨水、黑场人物 | 建立潜意识节拍 |
| 呼吸镜头 | 在高压段之间插入 2-5 秒的低动作镜头 | 让观众感到身体存在 |
| 空间放大 | 从狭窄极近切到巨大负空间远景 | 表现从压迫到心流 |

## 60 秒节奏建议

- 0-10 秒：以 2-4 秒镜头建立人物和压力场。
- 10-28 秒：穿插 0.6-1.2 秒快切组，连续出现触觉细节。
- 28-38 秒：把压力推到临界，允许更强的晃动、闪光和声音密度。
- 38-46 秒：节奏断崖，进入暴风眼，镜头变稳，声音抽空。
- 46-60 秒：超现实心流段，镜头稍长，运动更平滑，情绪从恐惧转为控制。

## 声音处理

每个镜头都要有 SFX，但不要让视频模型生成背景音乐。音乐、低频氛围和整体混音应在后期统一完成，避免分段生成时音乐断裂。
