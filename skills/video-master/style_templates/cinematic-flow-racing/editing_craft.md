# Editing Craft

## 剪辑原则

剪辑要像心理压力曲线，而不是平均分镜表。前半段用断裂和压缩制造不安，临界段用快切堆叠触觉信息，暴风眼段用突然留白制造反差，结尾用更长、更稳定的镜头完成心流释放。

## 可复用技巧

| 技巧 | 用法 | 目的 |
| --- | --- | --- |
| 硬切 | 在机械细节、眼神、仪表、手部动作之间直接切换 | 让观众感到没有缓冲 |
| 冲击切 | 在声音峰值、光闪、动作撞点处切下一镜 | 制造临界压力 |
| 概念蒙太奇 | 把胜利、毁灭、孤独、群众、火焰、奖杯等非连续画面并置 | 快速建立宿命感和反差 |
| 空间破坏跳剪 | 在宏观空间推进和微观材质特写之间跳切 | 制造颠簸、幽闭和生理不适 |
| 主客观交叉剪辑 | POV、脸部极近、系统信号、反射残影互相撞击 | 表现外部机械暴走和内部情绪崩溃 |
| 声画错位 | 下一镜的声音提前进入，或上一镜的声音延迟离开 | 让镜头连接更紧 |
| J-cut/L-cut | 口播或 SFX 先于画面进入/滞后退出 | 保持心理连贯 |
| 节奏断崖 | 快切后突然进入长镜头和低声场 | 形成暴风眼 |
| 极动转极静硬切 | 临界点后不接爆炸，而接慢动作或静止中心 | 打破观众预期，进入心流 |
| 重复意象回环 | 反复出现眼神、红灯、手、仪表、雨水、黑场人物 | 建立潜意识节拍 |
| 呼吸镜头 | 在高压段之间插入 2-5 秒的低动作镜头 | 让观众感到身体存在 |
| 空间放大 | 从狭窄极近切到巨大负空间远景 | 表现从压迫到心流 |
| 超现实长镜头 | 用平滑慢镜头表现云海、虚空、环形轨道或抽象路径 | 把职业动作升华成神性秩序 |

## 60 秒节奏建议

- 0-10 秒：宿命与反差。用概念蒙太奇交织光荣与毁灭，节奏相对平稳但意象要锋利。
- 11-22 秒：感官压缩。设置一个“物理开关”，随后用空间破坏跳剪把宏观运动和微观材质撞在一起。
- 23-37 秒：意识拉扯。使用主客观交叉剪辑，把人物情绪、POV、反射残影和临界信号推到红线。
- 38-44 秒：跃迁。用极动转极静的硬切进入慢动作、光束、几何运动或暴风眼。
- 45-60 秒：神性心流。剪辑频率断崖式下跌，用超现实长镜头和巨大留白收束。

## 时间线排列逻辑

- 序幕不要急着进入动作，要先把“奇迹/毁灭”的价值冲突摆出来。
- 起跳段的关键不是速度，而是从宏观叙事突然进入微观体感。
- 高潮段的快切要有方向，所有镜头最终都应汇聚到一个具象临界物：红区、倒计时、监护曲线、断裂阈值、最后一拍。
- 跃迁段要反预期：临界点后接静止、升格或抽象意象，而不是继续堆动作。
- 终局段要敢于留白，避免用字幕或口播解释“心流”；让镜头长度和空间秩序自己说话。

## 声音处理

每个镜头都要有 SFX，但不要让视频模型生成背景音乐。音乐、低频氛围和整体混音应在后期统一完成，避免分段生成时音乐断裂。
